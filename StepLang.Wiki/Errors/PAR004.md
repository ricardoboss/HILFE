# Missing Condition Expression Exception

## Description

The condition expression is missing.

Example:

```step
if () {
    print("Hello World!")
}
```

or:

```step
while () {
    print("Hello World!")
}
```

## Remediation

Add a condition expression to the `if` or `while` statement.

Example:

```diff
- if () {
+ if (true) {
    print("Hello World!")
}
```

or:

```diff
- while () {
+ while (false) {
    print("Hello World!")
}
```
